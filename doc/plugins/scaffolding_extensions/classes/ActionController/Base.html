Two variables can be set that affect scaffolding, either as class variables (which specifies the default for all classes) or instance variables (which specifies the values for that class only).

  • scaffold_template_dir: the location of the scaffold templates (default: "#{File.dirname(FILE)}/../scaffolds" # the plugin’s default scaffold directory)
  • default_scaffold_methods: the default methods added by the scaffold function (default: [:manage, :show, :destroy, :edit, :new, :search, :merge, :browse] # all methods)
